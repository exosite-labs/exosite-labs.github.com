Exosite Labs

Exosite Labs is a github organization that we use to post libraries, helper functions/classes and worker applications (listed below) that we expect developers to use intact in their projects that use Exosite's online services.

JSON RPC Bindings

Library bindings for the full JSON RPC API.
  • cpponep - C++ bindings to Exosite One Platform API (JSON RPC) over HTTP
  • clronep - .NET bindings to Exosite One Platform API (JSON RPC) over HTTP
  • pyonep - Python bindings to Exosite One Platform API (JSON RPC) over HTTP
  • javaonep - Java bindings to Exosite One Platform API (JSON RPC) over HTTP

XMPP Chat

XMPP chat interface helper library.
  • pyexompp - This library supports an XMPP chat interface to the Exosite Data Platform
